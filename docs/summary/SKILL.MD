# Library Management System - Role Summary

This document outlines the roles and features of the Library Management System.

## Roles and Features

### 1. Admin
The Admin has the highest level of authority and oversees the system's core management tasks.
- **Manage Books:** Responsible for adding and managing the list of available books in the library.
- **User Management:** The only role authorized to create Librarian accounts.
- **System Oversight:** Can view the total number of student and librarian accounts currently registered in the system.

### 2. Librarian
Librarians handle the day-to-day operations of book circulation.
- **View Books:** Can view the list of available books.
- **Borrowing Records:** Responsible for recording book loans, including:
    - Borrower Information (Who borrowed the book)
    - Date Borrowed
    - Return Date

### 3. Student
Students are the primary users of the library services.
- **Notifications:** Receives automated notifications when they borrow a book.
- **Reminders:** Receives notifications regarding the due date for returning borrowed books.
